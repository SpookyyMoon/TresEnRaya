  1. Objetivo del Juego:
    2.

    El objetivo es formar una línea horizontal, vertical o diagonal con tres símbolos iguales ('X' o 'O') antes que el oponente.

  2. Jugadores:
    3.
    • Jugador 1: Usa el símbolo X
    • Jugador 2: Usa el símbolo O
    • Se alternan los turnos, empezando por X
  3. Cómo Jugar:
    4.
    • Turnos alternados: Cada jugador coloca su símbolo en una casilla vacía por turno.
    • No se puede sobrescribir: Una casilla ocupada no puede modificarse
    • Gana el primer jugador que complete una línea de tres símbolos iguales.
    • Empate si el tablero se llena sin que nadie forme una línea.
